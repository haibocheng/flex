Installation

Before continuing you may wish to review the Flex SDK Release Notes.

This package includes a number of elements from Adobe that add additional capabilities to the Open Source Flex SDK. You should already have installed the Open Source Flex SDK, but if you have not it is available here. The Adobe® Flex™ SDK is a package that already includes everything found in the Open Source Flex SDK and this package, you may wish to install that instead.

The contents of this package are meant to be deployed on top of the contents of the Open Source Flex SDK. If the Open Source Flex SDK is located in Flex_Install_Dir, everything within this directory should be copied to Flex_Install_Dir. For example, after you have copied successfully you will find both READMEs located in the Flex_Install_Dir: Flex_Install_Dir/readme-addons.htm and Flex_Install_Dir/readme-open.htm.
