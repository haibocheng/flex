*** Adobe Flex Component Kit for Flash Professional

This folder contains the source files for the Flex Component Kit for Flash Professional. If you wish to install the Flex Component Kit for Flash Professional you can find the FlexComponentKit.mxp in the following directory:

<SDK install_dir>/frameworks/flash-integration/


For more information on using the Flex Component Kit for Flash Professional, see "Importing Flash CS3 Assets into Flex" in the Flex Developer's Guide at http://www.adobe.com/go/learn_flex3.
